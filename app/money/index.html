Money filter

A filter that appends a dollar sign to a valid number. We wrapped Angular's native currency filter so that we can properly display invalid inputs.

  • Currency 123.4: {{ 123.4 | money }}
  • Currency string "123.4": {{ "123.4" | money }}
  • Invalid Currency "abc": {{ "abc" | money }}

Money editing directive

A simple directive that makes a currency editable (uses the same directive as percent, decimal and integer).

The directive will broadcast a message when the edit starts and when it finishes so that your controllers and models can handle the edit properly.
