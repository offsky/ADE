AngularJS Datatype Editors (ADE)

ADE is a bunch of modules for displaying and editing various types of data, using AngularJS. The typical way that we have seen this done is to have hidden input boxes that toggle to visible when clicked. The important difference with ADE is that the input box is created and destroyed only when needed, so the DOM is not filled with unused elements. This can be a benefit when you have lots of editable things on the page.

ADE requiers jquery, bootstrap, and angular. You can find us on GitHub.

Download: ADE 1.2b

Dates


Date - Formats a unix timestamp as a date (Jan 1, 2013) and provides a popup calendar for editing.
Example:
{{datadate | validDate:'MMM d, yyyy'}}


Date + Time - Formats a unix timestamp as a date + time (Jan 1, 2013 11:45 am) and provides a popup calendar for editing.
Example:
{{datadate | validDate:'MMM d, yyyy h:mm:ss a'}}


Year - Formats a unix timestamp as a year (2013) and provides a popup calendar for editing.
Example:
{{datadate | validDate:'yyyy'}}


Time - Formats an integer as a time of the day and provides a popup widget for edting the time.
Example:
{{ datatime | time}}



Numeric


Money - Formats a decimal as currency, rounded to 2 decimal places witha $ in front.
Example:
{{ datacurrency | money }}


Decimal - Formats a decimal with a certain precision.
Example:
{{ datadec | decimal }}


Integer - Formats an integer.
Example:
{{ dataint | integer }}


Percent - Formats a decimal as a percentage with an appended percent sign.
Example:
{{ datapercent | percent}}


Duration - Formats a number as a time duration (4hrs 24mins).
Example:
{{ datalen | duration }}
