AngularJS Datatype Editors (ADE)

ADE is a collection of directives and filters for displaying and editing various types of data, using AngularJS. The typical way that we have seen this done is to have hidden input boxes that toggle to visible when clicked. The important difference with ADE is that the input box is created and destroyed only when needed, so the DOM is not filled with unused elements. This can be a benefit when you have lots of editable things on the page.

ADE requires jquery, bootstrap, and angular. You can find us on GitHub.

Install with Bower

bower install ADE

Or Download ZIP.
